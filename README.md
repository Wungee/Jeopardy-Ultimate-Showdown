## Overview

Interactive multiplayer Jeopardy game built with PHP, HTML, CSS, and the OpenTDB API.  
Core features:  
- Multiplayer, 2-4 players  
- Turn-based, game show animation  
- Dynamic OpenTDB rounds  
- Daily Doubles, Final Jeopardy betting  
- Streak bonus, player statistics tracking

## Instructions

- Clone repo and upload all files to your server
- Access `index.php` to start
- Play with friends and enjoy the experience!
